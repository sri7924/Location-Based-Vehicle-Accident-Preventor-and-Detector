# Location-Based Vehicle Accident Preventor and Detector

## Overview

An IoT-based vehicle safety system designed to prevent accidents and provide emergency alerts using GPS and GSM technologies.

## Features

- Obstacle Detection
- Accident Detection
- GPS Tracking
- Emergency SMS Alert
- Emergency Calling

## Hardware Used

- ESP32
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- GPS Module
- GSM Module
- MPU6050
- Vibration Sensor

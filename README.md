# Location-Based Vehicle Accident Preventor and Detector

## Overview

An IoT-based vehicle safety system designed to prevent accidents and provide emergency alerts using GPS and GSM technologies.

## Features

- Obstacle Detection
- Accident Detection
- GPS Tracking
- Emergency SMS Alert
- Emergency Calling
- Vehicle Safety Monitoring

## Hardware Used

- ESP32
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- NEO6M GPS Module
- SIM800L GSM Module
- MPU6050
- Vibration Sensor

## Repository Contents

- Project Presentation
- Hardware Prototype Image
- Working Procedure
- Hardware Requirements
- Future Enhancements

## Working

1. Detects obstacles using ultrasonic sensors.
2. Monitors accident conditions using sensors.
3. Tracks location using GPS.
4. Sends emergency alerts using GSM.
5. Improves vehicle safety and emergency response.

# Location-Based Vehicle Accident Preventor and Detector

## Overview
An IoT-based vehicle safety system designed to prevent accidents and provide emergency alerts with real-time GPS location tracking.

## Features
- Obstacle Detection using Ultrasonic Sensors
- Accident Detection using MPU6050 and Vibration Sensor
- GPS Location Tracking
- Emergency SMS Alerts
- Emergency Phone Calls
- Blynk IoT Monitoring

## Hardware Components
- ESP32
- Arduino UNO
- SIM800L GSM Module
- NEO6M GPS Module
- MPU6050
- HC-SR04 Ultrasonic Sensor
- Vibration Sensor

## Software
- Arduino IDE
- C++
- Blynk IoT

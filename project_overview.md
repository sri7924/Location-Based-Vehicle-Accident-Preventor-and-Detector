# Project Overview

Location-Based Vehicle Accident Preventor and Detector is an IoT-based vehicle safety project.

## Objectives

- Prevent vehicle accidents.
- Detect obstacles.
- Track vehicle location.
- Send emergency alerts.
- Improve road safety.

## Technologies Used

- Arduino UNO
- ESP32
- GPS
- GSM
- Ultrasonic Sensor
- MPU6050
- Blynk IoT
